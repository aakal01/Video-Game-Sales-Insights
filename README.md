# Video-Game-Sales-Insights

# Overview of The Repository
An end-to-end video game sales project from data cleaning, analysis, and visualization using Excel, Power Query, Power BI, and PowerPoint which is a example for data reporting insights to stakeholders.

# Project Overview
This project takes a comprehensive look at video game sales, analyzing performance across regions, genres, platforms, publishers, and time periods. The outcome is an interactive dashboard that makes it easy to explore trends and draw meaningful insights. As well as creating a powerpoint as a demo for reporting the insights to the stakeholders.

# Tools Used
Microsoft Excel,
Pivot Tables,
Power Query,
Power BI,
Microsoft PowerPoint.

# Recommendations
Prioritize Action and Shooter games when targeting North American and European markets.
Increase marketing and development efforts for Role-Playing games in Japan.
Use region-specific sales data to tailor game releases, advertising strategies, and platform selection.
Study the characteristics of top-performing platforms and publishers when planning future releases.
Continue monitoring changes in genre and platform preferences over time rather than relying on one global strategy.
